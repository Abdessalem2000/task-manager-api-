# SaaS Analytics Dashboard 📊 (In Progress)

This project is a comprehensive dashboard for SaaS analytics. Currently, the **Backend API** is fully functional, and the **Frontend** is under development.

## 🚧 Project Status: Under Development
* **Backend**: Completed (Node.js, Express, MongoDB).
* **Frontend**: Pending / In Progress.

## ✨ Completed Backend Features
* **User Authentication**: Secure login and registration using JWT.
* **Database Integration**: Connected to MongoDB for data persistence.
* **API Endpoints**: Full CRUD operations for analytics data.
* **Middleware**: Implementation of authentication and error handling.

## 🛠️ Tech Stack
* **Server-side**: Node.js & Express.js.
* **Database**: MongoDB (via Mongoose).
* **Security**: Password hashing with Bcrypt and JWT for sessions.

## 📦 How to Test the API
1. Clone the repository.
2. Run `npm install` to install backend dependencies.
3. Use a tool like **Postman** or **REST Client** to test the endpoints in the `src/routes` folder.
